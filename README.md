# Flight Radar Tracker

A real-time flight tracking application built with **React 18** and **Redux Toolkit**. This project utilizes **Leaflet** to visualize flight data on an interactive map and demonstrates advanced global state management.

![radar](https://github.com/user-attachments/assets/8a74dc91-1179-45eb-bf77-ced2411d842a)


## Key Features

* **Interactive Map:** Flight visualization using **Leaflet** and **React-Leaflet**.
* **Global State Management:** Efficient data handling with **Redux Toolkit**.
* **Pagination:** Seamless data navigation using `react-paginate`.
* **Data Fetching:** API integration via **Axios**.
* **Responsive UI:** Styled with **Bootstrap** for mobile compatibility.

## Tech Stack

| Category | Technologies |
| :--- | :--- |
| **Framework** | React 18 (Vite) |
| **State Management** | Redux Toolkit, React-Redux |
| **Map Library** | Leaflet, React-Leaflet |
| **Styling** | Bootstrap 5 |
| **Utils** | Axios, React-Paginate |
